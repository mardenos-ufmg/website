# Criar vetor de quartis

<div class="ref-description section level2">

Cria um vetor que mostra em que faixa dos quartis os elementos do vetor
original estão

</div>

<div class="section level2">

## Usage

<div class="sourceCode">

``` r
quartis(vec)
```

</div>

</div>

<div class="section level2">

## Arguments

-   vec:

    vetor de números a serem analisados

</div>

<div class="section level2">

## Value

vetor com a faixa quartílica de cada número

</div>

</div>
